# 🌐 Página Web – Curso de Front-End

Este projeto consiste em uma **página web desenvolvida utilizando HTML e CSS**, criada durante as aulas de um **curso de desenvolvimento front-end**.

O objetivo do projeto foi praticar a **estruturação de páginas web**, organização de conteúdo e estilização utilizando CSS.

⚠️ **Observação:**
Este projeto foi desenvolvido **acompanhando as aulas do curso**, sendo construído **junto com o professor** como parte do processo de aprendizado.

---

## 🚀 O que o projeto apresenta

A página simula um **site institucional de cursos de desenvolvimento front-end**, contendo:

* **Header fixo** com título do site e navegação
* **Menu de navegação** com links
* **Conteúdo principal** apresentando informações sobre um curso de front-end
* **Imagem responsiva** utilizando a tag `picture`
* **Botão de ação ("Saiba mais")**
* **Seção de conteúdos em grid**
* **Rodapé com informações de contato**

---

## 📚 Conceitos praticados

Durante o desenvolvimento deste projeto foram praticados conceitos como:

* Estruturação de páginas com **HTML semântico**
* Utilização das tags `header`, `main`, `article`, `aside` e `footer`
* Estilização com **CSS**
* Uso de **variáveis CSS (`:root`)**
* **Flexibilidade e organização de layout**
* **Grid Layout**
* **Responsividade com Media Queries**
* **Header fixo**
* **Efeitos hover em botões e links**

---

## 🛠️ Tecnologias utilizadas

* HTML
* CSS
